# 增加卷狗排行榜

## What to build

为登录用户提供全站刷题排行榜，按累计提交答案次数展示所有注册用户的学习活跃度，同时展示累计正确数和正确率。排行榜只公开用户名和聚合学习统计。

## Acceptance criteria

- [ ] 排行榜包含所有注册用户，零答题用户也可显示。
- [ ] 刷题数量定义为累计提交答案次数。
- [ ] 按刷题数量降序；相同时按累计正确数降序，再按用户 ID 升序。
- [ ] 返回排名、用户名、刷题数量、正确数和正确率。
- [ ] 当前登录用户在列表中有清晰但克制的视觉标记。
- [ ] 游客不参与排名，且页面明确提示登录后查看。
- [ ] 接口不返回密码哈希或其他敏感字段。
- [ ] 服务层排序和统计有自动化测试。

## Blocked by

None - can start immediately
