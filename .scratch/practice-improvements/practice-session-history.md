# 刷题会话支持上一题与状态回看

## What to build

将新题模式从只能向前的临时状态改为会话内可回看的答题流程。移除“跳过”操作，增加“上一题”；返回上一题时恢复用户当时的选项、判分结果和解析，不重复提交，也不改变已写入的进度。

## Acceptance criteria

- [ ] 刷题页不再显示“跳过”按钮。
- [ ] 非第一题显示可用的“上一题”，第一题时按钮禁用。
- [ ] 每道题在当前批次内独立保存已选答案和判分结果。
- [ ] 返回已答题目时保持选择、高亮、正确或错误状态及解析。
- [ ] 从历史题目继续前进时不会重复调用答题接口。
- [ ] 登录用户和游客行为一致。

## Blocked by

None - can start immediately
